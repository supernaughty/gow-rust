|-----------|----------|-----------|-------------|-----------------|
| T-04-06-01 | Tampering | patch path traversal | mitigate | strip_path normalizes paths; after stripping, join with a base directory (CWD) using Path::new(base).join(stripped) to prevent "../../../etc/passwd" traversal. Reject absolute paths in patch targets. |
| T-04-06-02 | Denial of Service | large diff input (similar) | accept | similar::TextDiff operates on loaded strings; bounded by available RAM. Files >100MB should be diffed with external tool. Document this limit. |
| T-04-06-03 | Tampering | patch atomic_rewrite failure | mitigate | atomic_rewrite only replaces target after successful transform; partial patch results are never written to the target path. Rejects written to .rej file (additive, not destructive). |
| T-04-06-04 | Information Disclosure | diff recursive directory walk | mitigate | WalkDir follows filesystem permissions; unreadable files produce stderr warning (not silent skip). No data exfiltration beyond the two specified directories. |
| T-04-06-05 | Elevation of Privilege | patch executing commands | accept | Neither diff nor patch executes shell commands. diffy::apply only performs text substitution. |
