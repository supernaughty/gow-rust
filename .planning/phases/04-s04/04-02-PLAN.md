    --- CUT IMPLEMENTATION (crates/gow-cut/src/lib.rs) ---

    Clap Args:
    - `-b` / `--bytes LIST`: byte ranges (e.g. "1-3,5")
    - `-c` / `--characters LIST`: character (Unicode codepoint) ranges
    - `-f` / `--fields LIST`: field ranges (1-indexed)
    - `-d DELIM` / `--delimiter DELIM`: field delimiter (default: TAB)
    - `--output-delimiter STRING`: output delimiter (default: same as -d)
    - `--complement`: invert selected range
    - `--only-delimited` / `-s`: only print lines with delimiter (for -f mode)
    - `files: Vec<PathBuf>` positional

    `fn parse_ranges(list: &str) -> Vec<Range<usize>>` — parse "1-3,5,7-" into inclusive ranges (convert 1-indexed to 0-indexed internally).

    `enum Mode { Bytes(Vec<Range>), Characters(Vec<Range>), Fields(Vec<Range>) }`

    For each input line (read_until b'\n', strip CRLF):
    - Bytes mode: select byte positions from raw bytes
    - Characters mode: collect chars(), select by codepoint index, rejoin to bytes
    - Fields mode: split on delimiter, select fields by index, rejoin with output_delimiter

    Complement: invert selection (select all indices NOT in ranges).

    Use gow_core::path::try_convert_msys_path for file args. Stdin fallback when no files given.

    --- UNIQ IMPLEMENTATION (crates/gow-uniq/src/lib.rs) ---

    Clap Args:
    - `-c` / `--count`: prefix each output line with count of consecutive occurrences (right-justified in 7-char field: format!("{:>7} {}", count, line))
    - `-d` / `--repeated`: only print lines that appear more than once
    - `-u` / `--unique`: only print lines that appear exactly once
    - `-i` / `--ignore-case`: case-insensitive comparison
    - `-f N` / `--skip-fields N`: skip first N fields (whitespace-delimited) before comparing
    - `-s N` / `--skip-chars N`: skip first N characters before comparing
    - `-w N` / `--check-chars N`: limit comparison to first N characters (after skips)
    - `-z` / `--zero-terminated`: use NUL instead of newline as line delimiter
    - `input: Option<PathBuf>` and `output: Option<PathBuf>` positional args

    `fn compare_key(line: &[u8], skip_fields: usize, skip_chars: usize, check_chars: Option<usize>, ignore_case: bool) -> Vec<u8>` — extract the portion of the line used for comparison.

    Main loop: `read_until(delim, &mut line)`, maintain `prev_line` and `count`. When the comparison key changes (or EOF), flush the accumulated group to output according to flags (-c/-d/-u). Use `Box<dyn BufRead>` for input and `Box<dyn Write>` for output.

    Integration tests for cut: bytes, chars, fields with custom delimiter, complement, only-delimited.
    Integration tests for uniq: basic dedup, -c count, -d repeated, -u unique, -i case-insensitive.
