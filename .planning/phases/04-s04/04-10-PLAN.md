<objective>
Close VERIFICATION.md truth #4: "tr handles character ranges (a-z) and character classes ([:alpha:])".

Purpose: The `expand_set` function in `crates/gow-tr/src/lib.rs` handles POSIX range notation (`a-z`) and escape sequences (`\n`, `\012`) but has NO branch for POSIX character class notation (`[:classname:]`). When `[` is encountered it falls through to the literal case, pushing `[` as a byte. Common usage like `tr -dc '[:digit:]'` produces wrong results — the `[`, `:`, `d`, `i`, `g`, `i`, `t`, `:`, `]` characters are treated as literals rather than as the digit class expansion.

This plan adds a `expand_posix_class` helper and inserts a detection branch in `expand_set` for `[:..:]` patterns. All existing range/escape/translate/delete/squeeze/complement behaviors remain unchanged.

Output: `crates/gow-tr/src/lib.rs` updated; 3 new integration tests pass; all 4 existing integration tests still pass; `cargo test -p gow-tr` exits 0.

```rust
fn expand_set(s: &str) -> Vec<u8> {
    let mut res = Vec::new();
    let chars: Vec<char> = s.chars().collect();
    let mut i = 0;
    while i < chars.len() {
        if chars[i] == '\\' && i + 1 < chars.len() {
            match chars[i + 1] {
                'n' => res.push(b'\n'),
                // ... other escapes and octal ...
                _ => res.push(chars[i + 1] as u8),
            }
            i += 2;
        } else if i + 2 < chars.len() && chars[i + 1] == '-' {
            // Range: e.g. a-z
            let start = chars[i] as u8;
            let end = chars[i + 2] as u8;
            for b in start..=end { res.push(b); }
            i += 3;
        } else {
            // Literal — '[' falls through here, pushed as 0x5B
            res.push(chars[i] as u8);
            i += 1;
        }
    }
    res
}
```

```rust
fn expand_posix_class(class: &str, res: &mut Vec<u8>) {
    match class {
        "alpha" => {
            for b in b'A'..=b'Z' { res.push(b); }
            for b in b'a'..=b'z' { res.push(b); }
        }
        "digit" => {
            for b in b'0'..=b'9' { res.push(b); }
        }
        "lower" => {
            for b in b'a'..=b'z' { res.push(b); }
        }
        "upper" => {
            for b in b'A'..=b'Z' { res.push(b); }
        }
        "space" => {
            res.extend_from_slice(&[b' ', b'\t', b'\n', b'\r', b'\x0c', b'\x0b']);
        }
        "alnum" => {
            for b in b'0'..=b'9' { res.push(b); }
            for b in b'A'..=b'Z' { res.push(b); }
            for b in b'a'..=b'z' { res.push(b); }
        }
        "blank" => {
            res.extend_from_slice(&[b' ', b'\t']);
        }
        "xdigit" => {
            for b in b'0'..=b'9' { res.push(b); }
            for b in b'A'..=b'F' { res.push(b); }
            for b in b'a'..=b'f' { res.push(b); }
        }
        _ => {} // unknown class — push nothing
    }
}
```

**Change 2: Insert a POSIX class branch in `expand_set`** between the escape-sequence branch and the range branch. The final structure of the while loop body must be:

```rust
while i < chars.len() {
    if chars[i] == '\\' && i + 1 < chars.len() {
        // ... existing escape handling unchanged ...
        i += 2;
    } else if chars[i] == '[' && i + 1 < chars.len() && chars[i + 1] == ':' {
        // POSIX character class: [:classname:]
        let class_start = i + 2;
        let mut class_end = None;
        let mut j = class_start;
        while j + 1 < chars.len() {
            if chars[j] == ':' && chars[j + 1] == ']' {
                class_end = Some(j);
                break;
            }
            j += 1;
        }
        if let Some(end) = class_end {
            let class_name: String = chars[class_start..end].iter().collect();
            expand_posix_class(&class_name, &mut res);
            i = end + 2; // skip past ':]'
        } else {
            // No closing :], treat '[' as a literal byte
            res.push(b'[');
            i += 1;
        }
    } else if i + 2 < chars.len() && chars[i + 1] == '-' {
        // Range: e.g. a-z
        // ... existing range handling unchanged ...
        i += 3;
    } else {
        res.push(chars[i] as u8);
        i += 1;
    }
}
```
