       enum Address { Line(usize), Last, Regex(Regex) }
       struct AddressRange { start: Option<Address>, end: Option<Address> }

           Substitute { pattern: Regex, replacement: String, global: bool, nth: Option<usize>, print: bool, case_insensitive: bool },

           Print,
           Quit,
       }

       - For each line: evaluate address match, execute command, build output
       - Default: print each line unless `-n`; `p` command forces print; `d` skips remaining commands and default print
       - Reconstruct output with original line endings

| T-04-04-03 | Tampering | sed -i without backup | accept | Atomic rewrite ensures original is only replaced after successful transform. Partial writes never corrupt target. |
