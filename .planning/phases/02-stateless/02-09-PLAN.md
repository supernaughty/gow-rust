<!-- GNU env -S escape table (RESEARCH.md Q7 lines 594-611) -->

| Sequence | Meaning |
|----------|---------|
| `\c` | stop processing the remainder of the string (ignore everything after) — outside double quotes only |
| `\f \n \r \t \v` | standard control chars |
| `\#` | literal `#` |
| `\$` | literal `$` |
| `\_` | outside quotes: token delimiter; inside double quotes: literal space |
| `\"` `\'` `\\` | literal quote / backslash |
| `${VAR}` | environment variable substitution (curly braces REQUIRED; `$VAR` is error) |
| `#` | comment, when at token start — rest of line dropped |

States: root → delimiter → unquoted | single_quoted | double_quoted | comment (+ backslash sub-states).

<!-- Command semantics (GNU env invocation) -->
```
env [OPTION]... [NAME=VALUE]... [COMMAND [ARG]...]

Options:
  -i, --ignore-environment    start with empty env
  -u, --unset NAME            remove NAME from env (may repeat)
  -C, --chdir DIR             change to DIR before exec
  -S, --split-string STRING   parse STRING per env -S spec, splice into argv
  -0, --null                  separate output entries with NUL instead of newline (only for listing mode)
  -v, --debug                 print verbose exec trace to stderr
  --                          end of options
```

<!-- Execution flow -->
1. Parse options.
2. Apply `-S STRING` → expand STRING into additional tokens and splice them at the current position in argv.
3. Build the child env: if `-i` → empty; else → current env, apply `-u NAME` removals, apply `NAME=VALUE` pairs from argv.
4. If there's a trailing COMMAND → spawn via `std::process::Command` (argv array, no shell string).
5. If no COMMAND → print env (newline or NUL per `-0`).

//! GNU `env -S STRING` split-string parser.
//!
//! State machine with 4 top states (delimiter, unquoted, single_quoted, double_quoted)
//! plus backslash sub-states. Supports ${VAR} expansion via caller-provided closure
//! (no direct env access — testable).
//!
//! Reference: RESEARCH.md Q7 lines 590-672. Security: input bounded by caller
//! (typical CLI max 10 KB); no recursion; 1-pass expansion (expanded values are
//! NOT re-parsed per Pitfall 6).

use std::fmt;

const MAX_INPUT_LEN: usize = 10 * 1024;

#[derive(Debug, PartialEq, Eq)]
pub enum SplitError {
    MissingBrace { at: usize },
    UnterminatedQuote { quote: char },
    InputTooLong,
}

impl fmt::Display for SplitError {
    fn fmt(&self, f: &mut fmt::Formatter<'_>) -> fmt::Result {
        match self {
            SplitError::MissingBrace { at } => write!(f, "only ${{VARNAME}} expansion is supported (missing brace at byte {at})"),
            SplitError::UnterminatedQuote { quote } => write!(f, "unterminated quote: {quote}"),
            SplitError::InputTooLong => write!(f, "input too long (> {MAX_INPUT_LEN} bytes)"),
        }
    }
}

/// Tokenize a GNU env -S string. `env_lookup` resolves `${VAR}` expansions.
pub fn split<F: Fn(&str) -> Option<String>>(
    input: &str,
    env_lookup: F,
) -> Result<Vec<String>, SplitError> {
    if input.len() > MAX_INPUT_LEN {
        return Err(SplitError::InputTooLong);
    }

    let bytes = input.as_bytes();
    let mut tokens: Vec<String> = Vec::new();
    let mut cur = String::new();
    let mut has_token = false;  // true once we start accumulating (handles empty current)
    let mut i = 0;

    #[derive(Clone, Copy)]
    enum S { Delim, Unquoted, Single, Double }
    let mut state = S::Delim;

    while i < bytes.len() {
        let b = bytes[i];
        match state {
            S::Delim => {
                match b {
                    b' ' | b'\t' | b'\n' | b'\r' | 0x0B | 0x0C => { i += 1; }
                    b'#' => {
                        // Comment: skip until newline or EOS.
                        while i < bytes.len() && bytes[i] != b'\n' { i += 1; }
                    }
                    b'\\' => {
                        // Escape in delimiter context: \c early exit, others start a token.
                        if i + 1 < bytes.len() && bytes[i + 1] == b'c' {
                            break;
                        }
                        // Enter unquoted with the escape, re-process at the same index.
                        state = S::Unquoted;
                        has_token = true;
                    }
                    b'\'' => { state = S::Single; has_token = true; i += 1; }
                    b'"'  => { state = S::Double; has_token = true; i += 1; }
                    b'$' => {
                        // Start a token with a ${VAR} expansion.
                        i = expand_var(bytes, i, &mut cur, &env_lookup)?;
                        state = S::Unquoted;
                        has_token = true;
                    }
                    _ => {
                        cur.push(b as char);
                        has_token = true;
                        state = S::Unquoted;
                        i += 1;
                    }
                }
            }
            S::Unquoted => {
                match b {
                    b' ' | b'\t' | b'\n' | b'\r' | 0x0B | 0x0C => {
                        tokens.push(std::mem::take(&mut cur));
                        has_token = false;
                        state = S::Delim;
                        i += 1;
                    }
                    b'\'' => { state = S::Single; i += 1; }
                    b'"'  => { state = S::Double; i += 1; }
                    b'\\' => {
                        // Handle backslash escape (unquoted).
                        if i + 1 >= bytes.len() {
                            cur.push('\\');
                            i += 1;
                            continue;
                        }
                        let n = bytes[i + 1];
                        match n {
                            b'c' => {
                                // End parse (flush current token, stop).
                                if has_token {
                                    tokens.push(std::mem::take(&mut cur));
                                }
                                return Ok(tokens);
                            }
                            b'_' => {
                                // Token delimiter.
                                tokens.push(std::mem::take(&mut cur));
                                has_token = false;
                                state = S::Delim;
                                i += 2;
                                continue;
                            }
                            b'f' => { cur.push('\x0C'); i += 2; }
                            b'n' => { cur.push('\n'); i += 2; }
                            b'r' => { cur.push('\r'); i += 2; }
                            b't' => { cur.push('\t'); i += 2; }
                            b'v' => { cur.push('\x0B'); i += 2; }
                            b'#' => { cur.push('#'); i += 2; }
                            b'$' => { cur.push('$'); i += 2; }
                            b'\\' => { cur.push('\\'); i += 2; }
                            b'"' => { cur.push('"'); i += 2; }
                            b'\'' => { cur.push('\''); i += 2; }
                            _ => { cur.push('\\'); cur.push(n as char); i += 2; }
                        }
                    }
                    b'$' => {
                        i = expand_var(bytes, i, &mut cur, &env_lookup)?;
                    }
                    _ => {
                        cur.push(b as char);
                        i += 1;
                    }
                }
            }
            S::Single => {
                // Single quotes: almost everything is literal EXCEPT \\ and \'.
                match b {
                    b'\'' => { state = S::Unquoted; i += 1; }
                    b'\\' if i + 1 < bytes.len() => {
                        let n = bytes[i + 1];
                        if n == b'\\' || n == b'\'' {
                            cur.push(n as char);
                            i += 2;
                        } else {
                            cur.push('\\');
                            i += 1;
                        }
                    }
                    _ => { cur.push(b as char); i += 1; }
                }
            }
            S::Double => {
                match b {
                    b'"' => { state = S::Unquoted; i += 1; }
                    b'\\' => {
                        if i + 1 >= bytes.len() {
                            cur.push('\\');
                            i += 1;
                            continue;
                        }
                        let n = bytes[i + 1];
                        match n {
                            b'_' => { cur.push(' '); i += 2; }  // in double quotes = literal space
                            b'f' => { cur.push('\x0C'); i += 2; }
                            b'n' => { cur.push('\n'); i += 2; }
                            b'r' => { cur.push('\r'); i += 2; }
                            b't' => { cur.push('\t'); i += 2; }
                            b'v' => { cur.push('\x0B'); i += 2; }
                            b'#' => { cur.push('#'); i += 2; }
                            b'$' => { cur.push('$'); i += 2; }
                            b'\\' => { cur.push('\\'); i += 2; }
                            b'"' => { cur.push('"'); i += 2; }
                            _ => { cur.push('\\'); cur.push(n as char); i += 2; }
                        }
                    }
                    b'$' => {
                        i = expand_var(bytes, i, &mut cur, &env_lookup)?;
                    }
                    _ => { cur.push(b as char); i += 1; }
                }
            }
        }
    }

    // Check for unterminated quotes
    if matches!(state, S::Single) { return Err(SplitError::UnterminatedQuote { quote: '\'' }); }
    if matches!(state, S::Double) { return Err(SplitError::UnterminatedQuote { quote: '"' }); }

    if has_token {
        tokens.push(cur);
    }
    Ok(tokens)
}

/// Expand `${VAR}` starting at `i` (bytes[i] must be b'$'). Writes the expansion
/// into `cur` and returns the new index (just past the closing `}`).
/// Returns Err(MissingBrace) if no `{` follows `$`.
fn expand_var<F: Fn(&str) -> Option<String>>(
    bytes: &[u8],
    i: usize,
    cur: &mut String,
    env_lookup: F,
) -> Result<usize, SplitError> {
    debug_assert_eq!(bytes[i], b'$');
    if i + 1 >= bytes.len() || bytes[i + 1] != b'{' {
        return Err(SplitError::MissingBrace { at: i });
    }
    // Find closing brace.
    let mut j = i + 2;
    while j < bytes.len() && bytes[j] != b'}' {
        j += 1;
    }
    if j >= bytes.len() {
        return Err(SplitError::MissingBrace { at: i });
    }
    let name = std::str::from_utf8(&bytes[i + 2..j]).unwrap_or("");
    let value = env_lookup(name).unwrap_or_default();
    cur.push_str(&value);
    Ok(j + 1)
}

//! `uu_env`: GNU `env` with full parity (-i, -u, -C, -S, -0, -v).
//! Child process spawned via std::process::Command argv array — NEVER a shell string (D-19b).
//!
//! Reference: RESEARCH.md Q7 (split-string), CONTEXT.md D-19.

mod split;

use std::ffi::OsString;
use std::io::Write;
use std::process::Command as StdCommand;

use clap::{Arg, ArgAction, Command};

pub fn uumain<I: IntoIterator<Item = OsString>>(args: I) -> i32 {
    gow_core::init();

    let matches = gow_core::args::parse_gnu(uu_app(), args);

    let ignore_env = matches.get_flag("ignore-environment");
    let null_sep = matches.get_flag("null");
    let verbose = matches.get_flag("debug");
    let chdir: Option<String> = matches.get_one::<String>("chdir").cloned();

    let unset_names: Vec<String> = matches
        .get_many::<String>("unset")
        .map(|iter| iter.cloned().collect())
        .unwrap_or_default();

    let split_string: Option<String> = matches.get_one::<String>("split-string").cloned();

    let mut raw_operands: Vec<String> = matches
        .get_many::<String>("operands")
        .map(|iter| iter.cloned().collect())
        .unwrap_or_default();

    // Apply -S: expand STRING into extra tokens and prepend to operands.
    // Per Pitfall 6, ${VAR} expansion happens AGAINST THE CURRENT ENVIRONMENT,
    // even when -i is also present — the expansion is at parse time, -i clears
    // the env for the child AFTER parsing.
    if let Some(s) = split_string.as_ref() {
        let lookup = |n: &str| std::env::var(n).ok();
        match split::split(s, lookup) {
            Ok(mut tokens) => {
                tokens.extend(raw_operands.drain(..));
                raw_operands = tokens;
            }
            Err(e) => {
                eprintln!("env: {e}");
                return 125;  // GNU convention: env returns 125 on its own failures
            }
        }
    }

    // Split operands into NAME=VALUE pairs vs COMMAND + ARGS.
    let mut assignments: Vec<(String, String)> = Vec::new();
    let mut cmd_idx: Option<usize> = None;
    for (i, op) in raw_operands.iter().enumerate() {
        if let Some(eq) = op.find('=') {
            // Before any command — assignment.
            let (name, value) = op.split_at(eq);
            if !name.is_empty() && name.chars().next().map(|c| c.is_alphabetic() || c == '_').unwrap_or(false) {
                assignments.push((name.to_string(), value[1..].to_string()));
                continue;
            }
        }
        cmd_idx = Some(i);
        break;
    }

    // Build the child env.
    let mut child_env: Vec<(String, String)> = if ignore_env {
        Vec::new()
    } else {
        std::env::vars().collect()
    };
    for name in &unset_names {
        child_env.retain(|(k, _)| k != name);
    }
    for (k, v) in &assignments {
        // Replace or append
        let found = child_env.iter_mut().find(|(key, _)| key == k);
        if let Some(entry) = found {
            entry.1 = v.clone();
        } else {
            child_env.push((k.clone(), v.clone()));
        }
    }

    match cmd_idx {
        Some(idx) => {
            let command_name = &raw_operands[idx];
            let command_args = &raw_operands[idx + 1..];

            if verbose {
                eprintln!("env: executing: {command_name}");
                for a in command_args {
                    eprintln!("env:           {a}");
                }
            }

            let mut cmd = StdCommand::new(command_name);
            cmd.args(command_args);
            cmd.env_clear();
            for (k, v) in &child_env {
                cmd.env(k, v);
            }
            if let Some(dir) = chdir.as_ref() {
                cmd.current_dir(dir);
            }
            match cmd.status() {
                Ok(status) => status.code().unwrap_or(127),
                Err(e) => {
                    eprintln!("env: {command_name}: {e}");
                    127
                }
            }
        }
        None => {
            // No command → print the environment.
            let stdout = std::io::stdout();
            let mut out = stdout.lock();
            let terminator: &[u8] = if null_sep { b"\0" } else { b"\n" };
            for (k, v) in &child_env {
                let line = format!("{k}={v}");
                out.write_all(line.as_bytes()).ok();
                out.write_all(terminator).ok();
            }
            0
        }
    }
}

fn uu_app() -> Command {
    Command::new("env")
        .arg(Arg::new("ignore-environment").short('i').long("ignore-environment").action(ArgAction::SetTrue).help("start with an empty environment"))
        .arg(Arg::new("unset").short('u').long("unset").action(ArgAction::Append).num_args(1).help("remove variable from the environment"))
        .arg(Arg::new("chdir").short('C').long("chdir").num_args(1).help("change working directory to DIR"))
        .arg(Arg::new("split-string").short('S').long("split-string").num_args(1).help("parse STRING as GNU env -S"))
        .arg(Arg::new("null").short('0').long("null").action(ArgAction::SetTrue).help("end each output line with NUL, not newline"))
        .arg(Arg::new("debug").short('v').long("debug").action(ArgAction::SetTrue).help("print verbose information for each processing step"))
        .arg(Arg::new("operands").action(ArgAction::Append).trailing_var_arg(true).allow_hyphen_values(true))
}

**Note on exit codes:** GNU `env` uses 125 for its own parse/option failures, 126 for "command found but not executable", 127 for "command not found". Ours: `split` error → 125, spawn failure → 127, child exit code → propagated. Clippy may want `.code().unwrap_or(127)` to be a `const`; leave as-is for clarity.
