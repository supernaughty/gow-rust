From RESEARCH.md Pattern 3 (seq scaled-integer arithmetic — VERBATIM, do not deviate):

```rust
fn decimal_places(s: &str) -> u32 {
    if let Some(dot_pos) = s.find('.') {
        (s.len() - dot_pos - 1) as u32
    } else {
        0
    }
}

fn seq_decimal(first_str: &str, inc_str: &str, last_str: &str, sep: &str) -> i32 {
    let first: f64 = first_str.parse().expect("validated by caller");
    let inc: f64 = inc_str.parse().expect("validated by caller");
    let last: f64 = last_str.parse().expect("validated by caller");
    let precision = decimal_places(first_str)
        .max(decimal_places(inc_str))
        .max(decimal_places(last_str));
    let scale: i64 = 10_i64.pow(precision);
    let mut cur = (first * scale as f64).round() as i64;
    let inc_scaled = (inc * scale as f64).round() as i64;
    let end = (last * scale as f64).round() as i64;
    let going_up = inc_scaled > 0;
    let stdout = std::io::stdout();
    let mut out = stdout.lock();
    let mut first_item = true;
    use std::io::Write;
    loop {
        if going_up && cur > end { break; }
        if !going_up && cur < end { break; }
        if !first_item { let _ = out.write_all(sep.as_bytes()); }
        first_item = false;
        if precision == 0 {
            let _ = writeln!(out, "{cur}");
        } else {
            let _ = writeln!(out, "{:.prec$}", cur as f64 / scale as f64, prec = precision as usize);
        }
        cur += inc_scaled;
    }
    0
}
```

#[command(name = "seq", about = "GNU seq — Windows port.",
          version = env!("CARGO_PKG_VERSION"),
          disable_help_flag = true, disable_version_flag = true)]
struct Cli {
    #[arg(long, action = ArgAction::Help)]
    help: Option<bool>,
    #[arg(long, action = ArgAction::Version)]
    version: Option<bool>,
    /// Use STRING as the separator between numbers (default: \n)
    #[arg(short = 's', long = "separator", default_value = "\n")]
    separator: String,
    /// Positional arguments: 1..=3 numbers (LAST | FIRST LAST | FIRST INC LAST)
    numbers: Vec<String>,
}
```

run(cli) logic:
1. Match `cli.numbers.len()`:
   - 0: print usage error to stderr, return 2 (no positional args)
   - 1: first="1", inc="1", last=numbers[0]
   - 2: first=numbers[0], inc="1", last=numbers[1]
   - 3: first=numbers[0], inc=numbers[1], last=numbers[2]
   - >=4: usage error, return 2
2. Parse first/inc/last as f64 — if parse fails, eprintln "seq: invalid floating point argument: '<arg>'" and return 1
3. If inc parses to 0.0 — eprintln "seq: invalid Zero increment value: '0'" and return 1
4. Determine precision via `decimal_places()` over the THREE input STRINGS (not parsed floats)
5. Apply scaled-integer loop from RESEARCH.md Pattern 3 — output via writeln! to stdout.lock()
6. Use `cli.separator` as the inter-value separator (NOT after the last value); but always end with a final newline if separator does not contain newline. GNU seq always emits a trailing newline regardless of -s.

Edge cases to handle correctly:
- `seq 5` (single arg): first=1, inc=1, last=5
- `seq 1 5` (two args): inc defaults to 1
- Negative inc: loop condition flips to `cur >= end`
- `seq 5 -1 1`: precision=0, integer fast path
- Empty range (e.g. `seq 5 1 1` going up but first > last): print nothing, exit 0

          version = env!("CARGO_PKG_VERSION"),
          disable_help_flag = true, disable_version_flag = true)]
struct Cli {
    #[arg(long, action = ArgAction::Help)]
    help: Option<bool>,
    #[arg(long, action = ArgAction::Version)]
    version: Option<bool>,
    /// Wrap width (default 80)
    #[arg(short = 'w', long = "width", default_value = "80")]
    width: usize,
    /// Break at the last space in the window when present
    #[arg(short = 's', long = "spaces", action = ArgAction::SetTrue)]
    spaces: bool,
    /// Count bytes rather than characters (Phase 10 always counts bytes — flag accepted for compat)
    #[arg(short = 'b', long = "bytes", action = ArgAction::SetTrue)]
    bytes: bool,
    files: Vec<String>,
}
```

run(cli) logic:
1. If `cli.width == 0`: eprintln "fold: invalid number of columns: '0'", return 1
2. If files.is_empty(): wrap_stream(stdin, stdout, width, spaces)
3. Else: for each file path: open file, wrap_stream, on error eprintln "fold: <path>: <err>" and exit_code=1
4. wrap_stream(reader, writer, width, spaces):
   - Read line by line (use BufRead::lines or bstr lines)
   - For each line: if line.len() <= width, emit line + "\n"
   - Else: emit substrings of length `width` (or shorter if -s and a space is within window)
   - With -s: scan back from position `width-1` to find a space byte; if found, break at space+1; if no space found, fall back to hard break at width
5. Phase 10 limitation: byte counting only (no Unicode width). Document as future enhancement. Tab stop and column counting (\b backspace, \r carriage return handling) are out of scope.
